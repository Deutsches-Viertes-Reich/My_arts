# [Suwa's battleship rookie edition!]

## 制作時期
2025/12/27 ～ 2026/1/18

## 作品概要
この作品は、私が以前の課題で制作した「戦艦を作って戦うゲーム」をベースに、9ヶ月間のプログラミング1,2の学習の集大成として大幅にアップグレードさせた作品です。ブラウザゲーム風の親しみやすいグラフィックを採用しつつ、本格的なコマンド選択と多彩なカスタマイズ要素を盛り込みました。プレイヤーは戦艦、重巡洋艦、軽巡洋艦、航空母艦、駆逐艦といった艦種からベースを選択でき、種類ごとに異なるステータスや専用スキルを駆使して戦います。カスタマイズの幅を広げるため、パーツ構成は船体、主砲、副砲、対空、艦長、装甲、エンジン、特殊アイテム、スキルカセット、艦橋の全10種類を用意しました。SFとファンタジーが融合した独特な世界観の装備を多数実装しており、パーツの組み合わせは100京通り以上です。技術的な工夫として、大量のパーツデータをまとめて管理するために前身であるゲームからリストや辞書(特にリスト)を流用し、オーバーホールしました。敵艦を撃破して得た資金でガチャを引き、新たなパーツを入手して自分だけの最強艦隊を作り上げる楽しさを追求しています。ぜひ、自分なりの理想の軍艦を完成させてみてください。
